Talk presented at **ROSCon France 2024** about [colcon-in-container](https://github.com/canonical/colcon-in-container), a colcon extension that allows building and testing ROS packages inside a container, removing the need to match your development machine's OS with the target ROS distribution.

[View the project on GitHub](https://github.com/canonical/colcon-in-container)
